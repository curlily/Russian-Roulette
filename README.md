# Russian Roulette

A terminal-based Russian Roulette game written in Rust.

Pick a number between 1 and 6. One of them shuts down your computer. The rest let you live to pick again. Survive all five safe chambers and you win.

## How to play

```bash
cargo run
```

Pick a number. Hope for the best.

## Warning

This will force-shutdown your machine if you hit the wrong number. Save your work first (or don't, that's kind of the point).
